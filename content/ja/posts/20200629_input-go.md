## 1行の入力を受け取る
```go
import "fmt"

func main() {
    var a string
    fmt.Scan(&a)

    fmt.Println(a)
}
```

## 空白が入った入力を受け取る
`fmt.Scanf`を使うと良さそう。
#### 固定長の場合
```go
import "fmt"

func main() {
    var a string
    var b string
    var c int
    fmt.Scanf("%s %s %d", &a, &b, &c)

    fmt.Println(a, b, c)
}

// ↓結果イメージ↓
// aaa bbb 3
//
// aaa
// bbb
// 3
```

#### 任意の場合
条件ごとに異なった桁数のデータを取得したい時はfor文を使って読み込みを行うと便利。
```go
import "fmt"

func main() {
    var inputWordLength string
    var words

    fmt.Scan(&inputWordLength)
    words := make([]string, inputWordLength)
    for i:=0; i<inputWordLength; i++ {
        fmt.Scanf("%d", &words[i])
    }
}
```


## 複数行の入力を受け取る
`fmt.Scan`では空白が入った時の標準入力をそのまま受け取ることができず、空白の前までの文字列が保存されるようになる。

```go
import (
    "fmt"
    "os"
    "bufio"
    )

func main() {
    stdin := bufio.NewScanner(os.Stdin)

    stdin.Scan()
    input := stdin.Text()

    fmt.Println(input)
}

```
